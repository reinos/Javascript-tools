How to use

Part 1 - Setup

  1. Slider 1.0 maakt gebruik van het jQuery framework en jQueryUI framework. Je zal in totaal 3 files moeten toevoegen aan je document.
    <script type="text/javascript" src="js/jquery.xxx.js"></script>
<script type="text/javascript" src="js/jquery.ui.widgets.js"></script>
<script type="text/javascript" src="js/jquery.slider.js"></script>
  2. Voeg de volgende style toe aan je css 
    #slider{ display:none; margin:0; padding:0; }  VERPLICHT
#animatie_nr li.selected{ background:#000 !important; color:#fff; }
  3. Voeg de volgende html toe aan je document: 
    <ul id="slider">
	<li><img src="images/foto1.png"/></li>
	<li><img src="images/foto2.png"/></li>
	<li><img src="images/foto3.png"/></li>
</ul>

Part 2 - Activate

  1. Voeg in een script tag de volgende code 
    $(window).ready(function(){
	$('#slider').slider();
)};
    Note: Let hierbij op dat je 'Window' gebruikt en geen 'document', dit omdat we de slider pas willen gaan gebruiken als alle images zijn geladen.
    Note: Dit is de basis voor de slider.

Part 3 - Config

  1. $('#slider').slider({
		sliderType:'fade', //type fade meegeven voor de fade
		numbers:true, //nummer tonen waar je op kan klikken
		clickAnimationTime: 500, //hoelang de animatie duurt wanneer je een user interaction hebt
		animationTime : 800, //hoe lang duurt de animatie
		interval: 2000, //om de hoeveel m/s slide de slider
		arrows:{ //pijlen voor links en rechts
			left:{
				src:'images/buttons.png',
				style:{
					height:'50px',
					width:'31px',
					position:'absolute',
					left:'-6px',
					zIndex:'99',
					top:'80px'
				}
			},
			right:{
				src:'images/buttons.png',
				style:{
					backgroundPosition:'right top',
					height:'50px',
					width:'31px',
					position:'absolute',
					zIndex:'99',
					right:'-6px',
					top:'80px'
				}	
			}
		},
		onInit : function(e, ui){
			//callback voor de init 
		},
		onStart : function(e, ui){
			//callback voor de onStart 
		},
		onComplete : function(e, ui){
			//callback voor de onComplete 
		},
		onClick : function(e, ui) {
			//callback voor de onClick, user-interaction 
		}
});
