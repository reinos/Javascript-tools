How to use

Part 1 - Setup

  1. Slider 1.0 maakt gebruik van het jQuery framework en jQueryUI framework. Je zal in totaal 3 files moeten toevoegen aan je document.
    <script type="text/javascript" src="js/jquery.xxx.js"></script>
<script type="text/javascript" src="js/jquery.ui.widgets.js"></script>
<script type="text/javascript" src="js/jquery.slider.js"></script>
  2. Voeg de volgende style toe aan je css 
    #slider{ display:none; margin:0; padding:0; }  VERPLICHT
#animatie_nr li.selected{ background:#000 !important; color:#fff; }
#slider h1, #slider p, #slider a {display:none;}
#overlayContent .more{background:url(images/prefix.png) no-repeat 0 2px ;height:23px;padding:0 0 0 12px;color:#fff;display:block;margin:10px 0 0 0;font-size:12px;}
#slider{display:none;margin:0;padding:0;}
#overlayContent h1{font-size:16px;}
#overlayContent p{font-size:12px;}
#animatie_nr {background:#fff;-moz-border-radius:5px;-webkit-border-radius:5px;position:absolute;bottom:-3px;right:-3px;z-index:999999;height:30px;}
#animatie_nr li{background:url(images/EURO_bol_actief.png) no-repeat;border:0 !important;text-indent:-999999px;width:20px;height:20px;margin:5px 0 0 0;}
#animatie_nr li.selected{background:url(images/EURO_bol_inactief.png) no-repeat !important;}
  3. Voeg de volgende html toe aan je document: 
    <ul id="slider">
	<li>
		<h1>Pagina 1</h1>
		<p>lacinia ut massa. In hac habitasse plateroin justo risus, aliquam ut euismrem ipsum dolorroin justo risus, aliquam ut euismre</p>
		<a href="#1">lees meer...</a>
		<img src="images/foto1.png"/>
	</li>
	<li>
		<h1>Pagina 2</h1>
		<p>lacinia ut massa. In hac habitasse plateroin justo risus, aliquam ut euismrem ipsum dolorroin justo risus, aliquam ut euismre</p>
		<a href="#2">lees meer...</a>
		<img src="images/foto2.png"/>
	</li>
</ul>

Part 2 - Activate

  1. Voeg in een script tag de volgende code 
    $(window).ready(function(){
	$('#slider').slider();
)};
    Note: Let hierbij op dat je 'Window' gebruikt en geen 'document', dit omdat we de slider pas willen gaan gebruiken als alle images zijn geladen.
    Note: Dit is de basis voor de slider.

Part 3 - Config

  1. $('#slider').slider({
		overlay:{ //overlay
			status:true, //overlay status
			overlayStyle:{ //style van de overlay
				background:'black',
				color:'white',
				padding:'10px', 
				width:'200px', 
				margin:'5px'
			},
			show:'images/arrow_show_EURO.png', //show button
			hide:'images/arrow_hidden_EURO.png', //hidden button
			hidden:false // auto hide
		},
		sliderType:'fade', //type fade meegeven voor de fade
		numbers:true, //nummer tonen waar je op kan klikken
		clickAnimationTime: 500, //hoelang de animatie duurt wanneer je een user interaction hebt
		animationTime : 800, //hoe lang duurt de animatie
		interval: 2000, //om de hoeveel m/s slide de slider
		onInit : function(e, ui){
			//callback voor de init 
		},
		onStart : function(e, ui){
			//callback voor de onStart 
		},
		onComplete : function(e, ui){
			//callback voor de onComplete 
		},
		onClick : function(e, ui) {
			//callback voor de onClick, user-interaction 
		}
});
