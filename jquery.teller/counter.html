Usage

Add the javascript in your head

<script type="text/javascript" src="jquery.1.7.2.js"></script> 
<script type="text/javascript" src="jquery.counter.min.js"></script> 
<script type="text/javascript" >$(function(){$('#counter').counter();});</script>

Add a new div in your html with the following markup:

<div id="counter">672:5</div>
